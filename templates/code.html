Sudo Sudoku

Question?

What's the best way to cheat at Sudoku and look smart to that pretty girl on the train?

Answer

Just take a picture of it with Sudo Sudoku!


Sudo Sudoku is an app that uses computer vision algorithms coupled with OCR to digitize Sudoku boards, and then uses some home-grown AI logic to return the solved puzzle.

iMadeIT.nu

Eagle-eyed viewers may realize that both this site and that site have very similar layouts. I am no web designer!

iMadeIT is a program I ran with for the Northeastern Entrepreneurs Club that teaches students with any level of programming background the fundamentals of web development in Flask by creating a Facebook clone.

To provide a sandbox for students, and add to the "Hey Mom, look what I made!" factor, I created iMadeIT.nu, which, after a user created an account on the website, created an identical account on the server that they could SSH into, and host their web app on our domain.

In addition, the site serves as a one-stop source for information about events, activities, as well as the source code to the site itself, other smaller web-apps, and other useful python resources
